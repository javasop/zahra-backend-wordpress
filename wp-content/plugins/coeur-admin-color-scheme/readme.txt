== Description ==

This plugin allows you to give your Wordpress dashboard a fresh new look. Your admin panel will look clean and will allow you to focus on what's really important for you and your users. This colours scheme is set as default and the plugin also prevents users from changing it.

== Installation ==

Here are the instructions you will need to follow in order to get your pluging working.

Automatically

1. Go to Plugins > Add New
2. Search for "Coeur Admin Color Scheme". Make sure it's the right plugin, the author should be "Titouanc‚Äù.
3. Hit install now
4. Hit activate now. Once activated the new color scheme will automatically be selected as default.

Manually

1. Upload `coeur-admin` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress and that's it.

== Frequently Asked Questions ==

Will the color scheme be used as default for every user?

- Yes the admin color scheme will be used as default for every user. 

Will the users be able to change the color scheme?

- No, the users will no longer be able to change the color scheme in their profiles.
